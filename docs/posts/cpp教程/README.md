## 🎯 学习路径

### 初学者路线

```text
1. C++ 快速入门 ← 语法基础
   ├─ .h 和 .cpp 文件
   ├─ 常用标准库（vector, string, algorithm）
   ├─ 类和 RAII
   └─ extern "C" 详解

2. CMake 基础 ← 构建系统
   ├─ Hello CMake
   ├─ 生成动态库
   └─ 文件组织

3. CMake 实战 ← NAPI 应用
   ├─ 第三方库管理
   ├─ 依赖链接
   └─ NAPI 模块构建

4. 语法深入 ← 理解规则
   └─ CMake 语法特点
```

### 进阶开发者路线

```text
1. NAPI 开发
2. HarmonyOS 原生模块
3. 性能优化
4. 多平台构建
```

### 图形学路线

```text
1. EGL 基础 ← 图形环境
   └─ EGL 是什么、如何初始化

2. OpenGL ES 3 ← 绘制技术
   └─ OpenGL ES 3 vs EGL 区别

3. 图形学基础 ← 渲染原理
   ├─ 为什么用三角形（6顶点/面）
   ├─ 顶点着色器（定位）
   └─ 片段着色器（上色）
```

## 📖 使用指南

### 如何阅读本教程

1. **按需学习**：根据目录选择需要的章节
2. **循序渐进**：建议按照学习路线顺序学习
3. **动手实践**：每篇文章都有完整代码示例
4. **查漏补缺**：可作为速查手册使用

### 学习顺序建议

**快速上手**：

```text
C++ 快速入门 → CMake 基础 → NAPI 构建
```

**深入学习**：

```text

```

**专项学习**：

- 只学 C++：阅读 "C++ 快速入门"

- 只学 NAPI：阅读 "NAPI 模块构建详解"

## 🔧 开发环境

### 推荐配置

- **C++ 编译器**：GCC/Clang
- **CMake**：3.16 以上
- **编辑器**：VSCode / CLion / DevEco Studio
- **操作系统**：Windows / Linux / macOS / HarmonyOS

### 快速搭建

1. 安装 C++ 编译器（随操作系统自带或下载）
2. 安装 CMake
3. 创建第一个项目
4. 按照教程实践

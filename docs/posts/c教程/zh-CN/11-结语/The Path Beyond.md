[English version](/posts/c教程/en-US/11-Epilogue/The Path Beyond)

现在您可以自信地用 C 语言编写代码，接下来的步骤自然如下：

1. 系统编程 探索 Linux 内部结构、系统调用和内核模块。像《Linux 编程接口》这样的书或者你未来的“C 系统编程小书”都是完美的伴侣。

2. 编译器和语言工具 编写自己的解析器或字节码解释器。 C 使您能够精确地从头开始构建新语言。

3. 操作系统和嵌入式尝试构建一个微型操作系统（如 xv6），或使用裸机 C 语言对微控制器进行编程。您将看到 C 如何塑造固件世界。

4. 库和开源 为用 C 编写的开源项目做出贡献，从 SQLite 到 Redis 再到 Git。您将阅读世界一流的 C 语言并通过示例学习设计。

5. 用 C 构建您自己的 X 您可以构建您自己的数据库、HTTP 服务器、shell 或编译器。每一次都是一次重新应用您在这里学到的知识的新机会。

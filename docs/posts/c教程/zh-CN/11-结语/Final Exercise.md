在离开本书之前，请编写最后一个 C 程序。它不需要做任何花哨的事情，只是提醒你为什么喜欢从第一原则构建东西。

```
#include <stdio.h>
int main(void) {
    printf("I learned to think in C.\n");
    return 0;
}
```

编译它。运行它。微笑。你现在可以说机器的语言了。

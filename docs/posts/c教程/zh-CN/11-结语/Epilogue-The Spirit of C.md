[English version](/posts/c教程/en-US/11-Epilogue/Epilogue-The Spirit of C)

C 不仅仅是语法、指针或编译器。这是一种心态，教你思考机器实际上是如何工作的。

当你用 C 语言编写时，你就是在说计算机的母语。您逐字节地告诉处理器要做什么，您和硬件之间不会产生任何错觉。

您已经了解到：

- 每个变量在内存中都有一个精确的位置
- 每个函数调用在堆栈上都有一个成本
- 每一个指针都是小心的承诺
- 你写的每一行都会转化为指令和数据

C奖励那些深入思考的人，惩罚那些猜测的人。但当你掌握它时，你就会获得一种很少有语言可以比拟的自由。

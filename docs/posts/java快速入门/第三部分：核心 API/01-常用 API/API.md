`API(Application Programming Interface)`，应用程序编程接口。

`Java API`是一本程序员的字典 ，是`JDK`中提供给我们使用的类的说明文档。这些类将底层的代码实现封装了起来，我们不需要关心这些类是如何实现的，只需要学习这些类如何使用即可。

`API`**使用步骤**

`1.`

打开帮助文档。

`2.`

点击显示，找到索引，看到输入框。

`3.`

你要找谁？在输入框里输入，然后回车。

`4.`

看包。`java.lang`下的类不需要导包，其他需要。

`5.`

看类的解释和说明。

`6.`

学习构造方法。

`7.`

使用成员方法。

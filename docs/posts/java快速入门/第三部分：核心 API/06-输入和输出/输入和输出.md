```
在Java中，可以使用被称作流的通信系统来存储和检索信息，流是在java.io包中实现的，并由java.nio.file包做了改进。

字节流：用于处理字节、整数、和其他简单数据类型。字节流传递0~255的整数。很多类型数据都可以表示为字节格式，包括数字数据、可执行程序、Internet通信和字节码——Java虚拟机（JVM）运行的类文件。

字符流：用于处理文本文件和其他文本源。

在Java中，所有数据都是使用流读写的，流就像水流一样，将数据从一个地方带到另一个地方。

流是程序中的数据所经历的路径。输入流将数据从数据源传递给程序，而输出流将数据从程序 发送到某个目的地。

实际上，每种数据都可以表示为单个字节或一系列字节。

字符流是一种特殊的字节流，只能处理文本数据。它不同于字节流，因为Java字符集支持Unicode——该标准包含很多无法使用字节来表示的字符。

对于任何涉及文本的数据，都应使用字符流，这包括文本文件、网页以及其他觉的文本类型。
```

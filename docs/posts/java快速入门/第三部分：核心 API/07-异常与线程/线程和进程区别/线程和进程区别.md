进程
一个在内存中运行的应用程序。 每个正在系统上运行的程序都是一个进程；

线程
进程中的一个执行任务（控制单元）， 它负责在程序里独立执行。

一个进程至少有一个线程，一个进程可以运行多个线程，多个线程可共享数据。

1. **根本区别**：
    1. 进程是操作系统管理资源的基本单位，每个进程都有独立的内存空间和系统资源。
    2. 线程是操作系统调度的基本单位，同一进程的多个线程共享进程的资源。
2. **资源开销**：
    1. 进程的切换涉及保存和恢复大量的系统资源和上下文信息，开销较大。
    2. 线程的切换只需保存和恢复较少的上下文信息，开销较小。
3. **包含关系**：
    1. 一个进程可以包含多个线程，线程在进程内执行任务。线程是进程的一部分。
    2. 线程可以并发执行，使得程序可以同时处理多个任务，提高程序的执行效率。
4. **内存分配**：
    1. 进程之间相互独立，每个进程有自己的地址空间，不共享内存。
    2. 线程共享同一进程的地址空间，可以直接访问进程的全局变量和资源。
5. **影响关系**：
    1. 一个进程的崩溃通常不会影响其他进程，因为它们是相互独立的。
    2. 一个线程的崩溃可能导致整个进程崩溃，因为线程共享进程的资源。
6. **执行过程**：
    1. 进程有自己的运行入口、执行顺序和退出方式，可以独立运行。
    2. 线程必须依赖于进程，由进程创建和管理。线程可以并发执行，提高程序的响应速度和处理能力。

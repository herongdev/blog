stack：后进先出（LIFO）型结构

类Stack只定义了一个构造函数，这是一个默认构造函数，它创建一个空栈。
Stack s= new Stack();

Java类Stack包含用于操纵堆栈的方法；

可使用方法push()将新元素加入堆栈，这个方法将元素压入到栈顶：

可使用方法pop()将元素从堆栈中弹出，从而将其删除：

String s1 = (String) s.pop();
String s2  = (Sting) s.pop();

如果要获得栈顶元素，但并不将其从堆栈中弹出，可使用peek();

String s3 = (String) s.peek();

使用方法search()在堆栈中搜索元素：

int I = s.search("Two");如果找到，方法search()将返回该元素到栈顶的距离，否则返回-1。与涉及索引和列表的Java数据结构一样，类Stack在报告元素位置时，也以0开始。这意味着栈顶元素的位置为0，第4个元素的位置为3。

类Stack定义的最后一个方法是empty()，用于判断堆栈是否为空：

```
要在显示字符串要指定格式时，可以使用方法:
System.out.format()。

该方法接受两个参数：输出格式模板以及要显示的字符串。
如下例给显示的整数添加美元符号和逗号：
int accountBalance = 5005;
System.out.formate("Balance:$%,d%n",accountBalance)

格式字符串以百分符号%打头，后面跟一个或多个标志。格式字符串“%,d”显示十进制数，并将每3位用逗号分隔。格式字符串“%n”显示换行符。

double pi = Math.PI;
System.out.formate("%.11f%n",pi);
输出结果包含11位小数。
```

**应用场景：**返回的数据不关联实体类，只是统计结果
按照直属上级分组，查询每组的平均年龄，最大年龄，最小年龄。
并且只取年龄总和小于`500`的组。

select avg(age) avg_age,min(age) min_age,max(age) max_age from user
group by manager id
having sum(age) \<500

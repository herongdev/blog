mybatis-plus:
# MyBatis Mapper所对应的XML文件位置，请注意与使用的数据库匹配
# mapper-locations: classpath*:mapper/**/sqlserver/*.xml
mapper-locations: classpath*:mapper/**/oracle/*.xml,classpath*:mapper/**/dm/*.xml,classpath*:mappers/*.xml
global-config:
# 关闭MP3.0自带的banner
banner: false
db-config:
#字段策略 0:"忽略判断",1:"非 NULL 判断"),2:"非空判断"
# IGNORED：“忽略判断”，所有字段都更新和插入。
# NOT_NULL：“非 NULL 判断”，只更新和插入非NULL值。
# NOT_EMPTY：“非空判断”， 只更新和插入非NULL值且非空字符串。
# DEFAULT： 默认的，一般只用于注解里。
insert-strategy: not_null
update-strategy: not_null
selectStrategy: not_null
#数据库大写下划线转换
capital-mode: true
# 创建人、创建时间、版本号，是否删除等字段自动填充，默认使用下面实现类。
# 若不满足业务需求，可实现MetaObjectHandler接口进行自定义填充
meta-object-handler: com.aoip.orm.mybatis.handler.ModelMetaObjectHandler
logic-delete-field: delFlag
logic-delete-value: 1
logic-not-delete-value: 0
configuration:
#开启驼峰功能
map-underscore-to-camel-case: true
jdbc-type-for-null: 'null'
# mysql, oracle, dm 等，请注意与使用的数据库匹配
database-id: dm
#log-impl: org.apache.ibatis.logging.log4j2.Log4j2Impl
log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
这是一个针对MyBatis-Plus的配置，MyBatis-Plus是MyBatis的增强工具，主要通过增加一些实用功能来简化MyBatis的操作。下面我会逐个解释这个配置中的各个部分：

1. mapper-locations: 这是配置mapper XML文件的位置。在这里，配置了3个位置，分别对应oracle数据库的mapper，达梦数据库的mapper以及通用的mapper。MyBatis会在这些位置找到对应的XML文件，并将它们加载到应用中。
2. global-config: 这是全局配置，影响所有的mapper。
3. banner: 关闭MyBatis Plus的启动banner。
4. db-config: 数据库配置项，针对数据库操作的配置。
5. insert-strategy, update-strategy, selectStrategy: 字段策略，定义了插入、更新、选择时的字段策略，可选的值有IGNORED、NOT_NULL、NOT_EMPTY、DEFAULT，对应的策略分别是“忽略判断”、“非NULL判断”、“非空判断”和“默认策略”。
6. capital-mode: 是否启用大写下划线模式，如果启用，MyBatis Plus会将数据库表和字段名映射为大写并用下划线分隔。
7. meta-object-handler: 自动填充处理器，定义了自动填充创建人、创建时间、版本号以及是否删除等字段的实现类。
8. logic-delete-field: 逻辑删除字段，定义了用于标记记录是否被删除的字段名。
9. logic-delete-value and logic-not-delete-value: 逻辑删除和未删除的标记值，分别是1和0。
10. configuration: 这是MyBatis的核心配置，它包含了很多关于MyBatis行为的配置项。
11. map-underscore-to-camel-case: 是否启用驼峰命名法，如果启用，MyBatis会将数据库中的下划线命名方式映射为Java的驼峰命名方式。
12. jdbc-type-for-null: 定义了将null值映射到哪种JDBC类型。
13. database-id: 数据库标识符，这用于加载对应数据库的SQL映射语句。
14. log-impl: 日志实现，定义了MyBatis使用的日志实现类，这里配置的是将日志输出到标准输出。

这个配置中包含了许多高级功能，例如字段策略、逻辑删除、自动填充等，它们都是MyBatis Plus为了简化MyBatis操作而提供的增强功能。这些功能在大多数应用中都非常有用，可以大大提高开发效率。

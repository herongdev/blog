Java的版本和SDK（Software Development Kit，软件开发工具包）版本通常是一致的。例如，Java 8对应的SDK就是Java SDK 1.8，Java 11对应的SDK就是Java SDK 11，以此类推。在早期的Java版本中，主版本号前面会有一个“1.”的前缀，但从Java 9开始，官方取消了这个前缀，所以现在通常直接用主版本号来表示。

在讨论Java版本时，你可能会遇到几个相关的术语，包括JDK（Java Development Kit，Java开发工具包）、JRE（Java Runtime Environment，Java运行时环境）、和SDK（Software Development Kit，软件开发工具包）。这些术语中，JDK和SDK基本上是可以互换的，但JRE是指Java程序运行所需要的环境。
这里的对应关系简单列举如下：

- Java 1.6 对应 JDK 1.6 或者称为 Java 6
- Java 1.7 对应 JDK 1.7 或者称为 Java 7
- Java 1.8 对应 JDK 1.8 或者称为 Java 8
- Java 9 对应 JDK 9
- Java 10 对应 JDK 10
- Java 11 对应 JDK 11
- ...以此类推

在实际使用中，通常我们会说 JDK 8, JDK 11 等来指代具体的 Java 版本。

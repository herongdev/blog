2. `maven`：包管理器
使用`JDK`开发`Java`程序的缺点在于，这是一组命令行工具，因此没有用于编辑程序、将其转换为`Java`类并进行测试的图形用户界面。命令行是用于输入文本命令的提示符，在`Windows`中为程序“命令提示符”。

- 按两下`shift`可搜索类，资源，配置项，方法等
- 按`ctrl+f`在本页==按照文本的内容查找==
- 按`ctrl+shift+f`在全局==按照文本的内容查找==

`JShell`**脚本工具是**`JDK9`**的新特性**
什么时候会用到 `JShell` 工具呢，当我们编写的代码非常少的时候，而又不愿意编写类，`main`方法，也不愿意去编译和运行，这个时候可以使用`JShell`工具。

启动`JShell`工具，在`DOS`命令行直接输入`JShell`命令。
接下来可以编写`Java`代码，无需写类和方法，直接写方法中的代码即可，同时无需编译和运行，直接回车即可。

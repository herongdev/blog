**二进制**

`0`

**字节**
字节是我们常见的计算机中最小存储单元。计算机存储任何的数据，都是以字节的形式存储，右键点击文件属性，
我们可以查看文件的字节大小。

`8`

个`bit`（二进制位） `0000-0000`表示为`1`个字节，写成`1 byte`或者`1 B`。

- 8 bit = 1 B
- 1024 B =1 KB
- 1024 KB =1 MB
- 1024 MB =1 GB
- 1024 GB = 1 TB

**进入**`DOS`**操作窗口**

打开`DOS`命令行后，看到一个路径 `c:\user` 就表示我们现在操作的磁盘是`c`盘。

`Java`的数据类型分为两大类：

- 基本（`Primitive`）数据类型：包括 整数 、 浮点数 、 字符 、 布尔 。
- 引用（`Reference`）数据类型：包括 类 、 数组 、 接口 。

基本数据类型在声明是会先分配内存空间，目前共有四类八种基本数据类型，即：
字符（`Char`）类型，布尔（`Boolean`）类型，浮点（`FloatingPoint`）类型，整数（`Integer`）类型

 ;
 其中浮点类型包括单精度（`float`）浮点数和双精度（`double`）浮点数；
其中整数类型包括字节型（`byte`），短整型（`short`），整型（`int`）和长整型（`long`）；

布尔(Boolean)类型的变量用于关系运算的判断或逻辑运算的结果。

**字符类型**
在Java中，字符数据类型为char，它是一种使用16位(16-bit)二进制数所表示的Unicode字符。
特别要注意的是，字符数值是用单引号标注单个字符，字符串以双引号标注。

**练习**
定义所有基本数据类型的变量，代码如下：

public class Variable {
public static void main(String[] args){
//
`定义字节型变量`

byte b = 100;
System.out.println(b);
//
`定义短整型变量`

short s = 1000;
System.out.println(s);
//
`定义整型变量`

int i = 123456;
System.out.println(i);
//
`定义长整型变量`

long l = 12345678900L;
System.out.println(l);
//
`定义单精度浮点型变量`

float f = 5.5F;
System.out.println(f);
//
`定义双精度浮点型变量`

double d = 8.5;
System.out.println(d);
//
`定义布尔型变量`

boolean bool = false;
System.out.println(bool);
//
`定义字符型变量`

char c = 'A';
System.out.println(c);
}
}

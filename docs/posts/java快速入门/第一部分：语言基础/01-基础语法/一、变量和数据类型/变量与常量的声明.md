变量是具备名称的一块内存空间，用来存储可变动的数据内容。当程序需要存取某个内存在的内容时，就可以通过变量名称将数据从内存中取出或写入这个内存空间。在使用变量前，必须对变量进行声明，`Java`语言的变量声明语句可分为“数据类型”与“变量名称”两部分。声明语句语法如下：
数据类型 变量名称；`//` 符号“；”表示语句的结束，如：

int num1=30;
int num2=77;
这时`Java`会自动分配`4`字节的内存空间给变量`num1`和`num2`，它们的存储值分别为`30`和`77`。当程序运行过程中需要存取这块内存空间的内容时，就可以直接使用变量名称`num1`和

`num2`

进行存取，如图：

|   |   |   |
|---|---|---|
|内存位置|值|变量名称|
|`1024`|`30`|`num1`|
|`1028`|`77`|`num2`|

当财时要声明多个相同数据类型的变量时，可以使用逗号“，”来分隔变量名称，例如：
`long apple,banana;`
完成变量的声明后，有些变量可根据需要赋初始值，就是在变量声明的语句中加入初值的设置，语法如下：
**数据类型** **变量名称** `=` **初值；**
例如：

int apple = 5;
boolean a = true,b = false;//
同时声明多个`boolean`类型的变量并赋初值
在赋初值时，注意数据类型为字符和浮点数的赋值。给`char`数据类型赋初值有三种方法：字符、`Unicode`码以及`ASCII`码，其中初值为字符、`Unicode`码格式时，必须在字符的左右两边各加上单引号“`'`”，如：

char apple = '@';
char apple = '\u0040';
char apple = 64;//
用‘`@`’字符对应的`ASCII`码`64`来赋初值，`ASCII`码用十进制数来表示

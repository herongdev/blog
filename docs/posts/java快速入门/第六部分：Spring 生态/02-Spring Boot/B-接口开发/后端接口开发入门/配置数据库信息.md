将`src/main/resources/application.properties`改名为`application.yml`

![Exported image](Exported%20image%2020260702230435-0.png)

内容如下：

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/constructional_engineering?useUnicode=true&serverTimezone=Asia/Shanghai&characterEncoding=utf8&autoReconnect=true&useSSL=false&allowMultiQueries=true
    username: root
    password: Hr123456
    driver-class-name: com.mysql.cj.jdbc.Driver
mybatis:
  mapper-locations: classpath:mapper/*Mapper.xml
  type-aliases-package: com.test.elasticsearchdemo.domain
constructional_engineering
==为数据库名称；==

运行，这时访问`localhost:8080`，显示`spring boot` 默认`404`页

![Exported image](Exported%20image%2020260702230437-1.png)

- `controller`展示 定义接口路径和调用`service`
- `service` 处理业务逻辑 数据库数据
- `mapper`定义操作数据库动作，命名
- `mapper.xml`执行`mapper`里定义的动作的`sql`语句，与数据库交互
- `entity` 定义类，与数据库类型保持一致

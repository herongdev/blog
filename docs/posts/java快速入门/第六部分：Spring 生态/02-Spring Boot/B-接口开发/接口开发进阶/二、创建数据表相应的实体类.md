类名和表名建议保持一致，此外字段和字段的类型必须一致；

- 类的注解主要是：
    - `Lombok`
    - 和`swagger`
    - 还有`mybatisplus`的数据表
- 其中字段的注解主要是：
    - 数据表中对应字段和`swagger`的注释
    - 要注意对`id`的特殊处理
